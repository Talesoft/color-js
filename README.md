A color manipulation and conversion library.

Installation
------------

```bash
npm i @talesoft/color
```

Usage
-----

Color JS makes color manipulation a breeze.

```javascript
import { Color, Colors } from '@talesoft/color';

const red = Colors.RED;

red.fadeOut()

```
